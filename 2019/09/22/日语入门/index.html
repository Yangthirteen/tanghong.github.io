10.第25页

语法解释

1.名 は 名 です：相当于汉语的’’…..是…….’’,”は”是主语部分，”です”是谓语部分。助词””用于提示主题，读作””

2.名 は 名　では　ありません：”……不是……”,”では”有时会读作”じや（小写 ）”

3.疑问句及其应答

  • 名 は 名　ですか：”…..是…. 吗？”，句尾不用”?”
  • 应答：
    • 肯定：”はい”、”はい　そうです”
    • 否定：”いいえ”、”いいえ　ちがいます”
    • 不知道：”(わ)分かりません”

4.名 の 名：前面的名词是后面名词从属的机构、国家或属性（不论名词之间是什么关系，一般加”の”）

表达及词语讲解

1.人称：

（1）”わたし”、”あなた”、”＊あの　（ひと）人”

  • 在不知道对方的姓名又必须得打招呼的情况下，才会使用第二人称”あなた”,因为用”あなた”来称呼对方，有时会显得很不礼貌

（2）”~さん”

  • 不分男女，都在其姓后面加”さん”
  • “さん”只用于他人姓名后面，不用于自己姓名后面
  • 用”姓+さん”来代替”あなた”
  • 称呼小孩的时候，一般在其姓名后面加”＊ちや(小写)ん”
  • 对于自己年龄相当，或者比自己年轻的男性，有时使用”＊（くん）君”来称呼

2.省略（第一、二人称主语的省略）

  • 一般省略通过上下文或者场景可以明白的部分

语法解释

1.これ/それ/あれ　は～です

  • これ：距离说话人较近
  • それ：距离听话人较近
  • あれ：与两者都比较远

2.だれですか/(なん)何ですか

  • だれ：不知道是谁时使用（谁）
  • (なん)何：不知道是什么东西时使用（什么）

3.～の～

4.この/その/あの～は～です

5.どれ/＊どの

  • 哪个：单独使用时用どれ、修饰名词时使用どの

2019.10.23

1.”あります”和”います”：表示事物的存在

  • “あります”用于不具有意志的事物
  • “います”用于具有意志的事物
  • ～に　～が　あります/います：有
  • ～は　～に　あります/います：在
  • 可用～は　どこに　ありますか/いますか代替～は　どこですか

2.～と～：并列（和）

3.上（うえ）/＊下（した）/＊前（まえ）/＊後（うし）ろ/隣（となり）/中（なか）/外（そと）

  • 表示具体位置时：名词+の＋～
  • 汉语中的”椅子上”之类的词语，日语中”いすの　上（うえ）”，”の”不可省略

4.ね：确认

5.疑问词+も+动词（否定）：表示全面否定

6.日语中的”上”只表示垂直上方的范围

7.＊ええと：表示被问到某事的时候，思考该如何回答时候说的话

8.”＊ご家族”,”＊ご兄弟”,”＊ご両親”：礼貌说法

9.＊一人（ひとり）暮らし：一个人生活

2019.10.30

1.动词1

  • 不以る结尾
  • 词尾是る，但其前的假名不是い/え段假名

2.动词2：以る结尾，前面是い/え段假名

3.动词3：する，前面必须要有两个汉字；来（く）る

4.动词变为连用形

  • 动词1：词尾变为同一行的い段假名（90%的连用形都是这么变化，区分不了动词1与动词2的连用形时常常是动词1的连用形），然后加ます
  • 动词2：去る＋ます
  • 动词3：する改为します；来る 变为 来（き）ます

2019.10.30

1.今（いま）　～時（じ）　～分（ふん）です

  • 表示现在的具体时间，询问用”何時（なんじ）”
  • 表示具体时间的词前面还可以加上”＊午前（ご　ぜん）”、”＊午後（ごご）”
  • 半点不可用　半（はん）

2.时态

  • 现在的习惯性动作、状态和为来的动作、状态：～ます
  • 否定形式：～ません
  • 过去的动作：～ました
  • 否定形式：～ませんでした
  • 疑问句在句尾加か

3.名[时间] に 动

  • 表示动作发生的时间时，要在具体的时间词语后面加上助词に
  • 叙述包含数字的时间时后续助词に，今日等词不加，星期后一般加，也可以不加

4.名[时间] から 名[时间] まで 动

  • 表示某动作发生的某个期间

5.いつ 动 ますか

  • 询问某动作或事态进行的时间
  • 询问很具体，在表示时间的词语后面加に，还要在句尾加ますか
  • 询问持续性动作或事态的起点呼终点时，使用”いつから”、”何曜日（なん　よう　び）です”

6.は：表示对比，发音略微重一点

7.~です：部分已经明确的内容，回答问题时不需要重复

8.~ごろ：表示时间大致范围

2019.11.11

1.名[场所] へ　动：表示移动的动词时，移动行为的目的地使用助词　へ，读作　え

2.名[场所] から 动词：移动的起点使用助词から

3.名[人] と 动：共同做某事的对象

4.名[交通工具] で 动：交通手段使用助词 で，步行使用 *歩（ある）いて

5.名[场所] から 名[场所] まで 动：移动的范围，起点和终点

6.に/で/へ/から/まで/と　＋　は：构成复合形式

7.*たしか：记忆不完全，好像是，大概是

8.です用法2：省略

9.家

  • 家（いえ）：建筑物的家
  • うち：我家，也指家人

10.*まっすぐ：表示“笔直的意思”，可以用来表示，“不顺路去别处，径直”

11.提问月份：何（なん）月（ぐつ）ですか

12.提问日期：何（なん）日（にち）ですか

2019.11.18

1.名 を 动：动作的对象用组词を，读作お

2.名[场所]で 动：动作的场所用组词で

  • 注意：存在场所：に；进行场所：で

3.名 か 名：相当于或者

4.名 を　ください：可用于花钱购物/不花钱索取（命令语气，对长辈不合适）

5.‘何’的读法：なん/なに

  • 根据后续音节不同，发生变化
  • 最基本读音：なに
  • 在组词の以及量词钱读成なん
  • 在で钱两种读法都可

6.*じゃあ：接过话题或者中断谈话时使用，比较随便的说法，相当于“那么”，正式说法 *では

7.黑板上写字：不用で，使用に

补充：考试会有改错题，先判断正误，再改错

2019.11.25

1.名[工具] で 动：交通工具，其他手段以及原材料

2.名1[人] は 名2[人] に 名3[物] を あげます

  • 给……..什么东西

3.名1[人] は 名2[人] に 名3[物] を もらいます

  • 得到……什么东西

4.名[人] に 会（あ）います：见到…..谁

5.よ：用于提醒

6.～から　もらいます：从某人那里接受

  • 给与的一方是“学校”之类的组织

7.さっき 和 *たった　今（いま）：刚刚

  • 表示离现在很近的过去，后续动词要用过去形式
  • 离过去非常近用第二个
  • 稍微前一点用第一个

8.名词+（を）お願（ねが）いします

  • 请求对方做某事

9.同一个意思，两种形式的单词

  • 即：自动词（带宾语）/他动词（不带宾语）
  • 前者为动词1
  • 后者为动词2

2019.12.02

1.名+は+一类形+です：形容词

  • 以い结尾的形容词，做谓语时要在后面加上です

2.一类形 做谓语时的否定形式

  • 将词尾的い变为く再加上ないです或者ありません
  • いいです的否定形式是よく　ないです　或者 よく　ありません
    • いい用作变化的词根是　良（よ）い

3.一类形 做谓语的过去形式

  • 将词尾的い变为かった再加です
  • 过去形式的否定形式：将词尾的い变为 く　なかったです　或者　く　ありませんでした
  • いいです的过去形式是 よかったです；　过去否定形式是　よく　ありませんでした

4.一类形+名：一类形直接修饰名词，不能加の

5.をー＞は：を前面的名词作话题，或者进行对比的时候，使用は，原来的を必须去掉

6.あまり＋一类形/动（否定）

  • あまり 与后面的否定形式呼应，表示程度不高

7.表示程度的副词

  • とでも、たいへん、＊少（すこ）し、＊ちょっと
  • あまり、全然（ぜんぜん）：要与后面的否定形式呼应使用

8.多（おお）い、少（すく）ない

  • 不能单独修饰名词
