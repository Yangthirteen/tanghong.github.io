  1. select:选择

    (1)select column_name,column_name from table_name;

    (2)select * from table_name;

  2. select distinct:选择并且去除重复

    select distinct column_name,column_name from table_name;

  3. where:过滤

    select column_name,column_name from table_name where column_name operator value;

= 等于
<> 不等于。注释：在 SQL 的一些版本中，该操作符可被写成 !=
> 大于
< 小于

  1. and & or:基于一个以上的条件进行过滤

    (1)如果第一个条件和第二个条件都成立，则 AND 运算符显示一条记录。

    (2)如果第一个条件和第二个条件中只要有一个成立，则 OR 运算符显示一条记录。

  2. order by:对结果集排序

    （1）默认升序，若要降序，使用desc关键字

    （2）select column_name,column_name from table_name order by column_name,column_name asc|desc;

    (3)前面的条件优先排序

  3. insert into:插入新数据

    （1)insert into table_name values (value1,value2,value3,…);

    (2)insert into table_name (column1,column2,column3,…) values (values1,value2,value3,…);

  4. update:更新记录

    update table_name set column1=value1,column2=value2,… where some_column=some_value;

  5. delete:删除记录

​ (1)delete from table_name where some_column=some_value;

​ (2)删除所有记录:

​ delete from table_name; or delete * from table_name;

  1. select top, limit, rownum

    (1)select top:规定要返回的记录的数目(mysql支持limit语句)

    (2)mysql:select column_name(s) from table_name limit number;

  2. like:在where子句中搜索列中的制定模式

    (1)select column_name(s) from table_name where column_name like pattern;

  3. 通配符:用于代替字符串中的其他任何字符

    (1)%:代替0个或者多个字符

    (2)_:代替一个字符

    (3)[charlist]:字符列中的任何单一字符

    (4)[^charlist]或[!charlist]:不在字符列中的任何单一字符

  4. in:在where子句中规定多个值

    (1)select column_name(s) from table_name where colimn_name in (value1,value2,…);

  5. between:选取介于两个值之间的数据范围内的值

    (1)slect column_name(s) from table_name where column_name between value1 and value2;

    (2)选取不在其范围使用not between

  6. as:别名

    (1)select column_name as alias_name from table_name;

    (2)select column_name(s) from table_name as alias_name;

  7. 连接(join):将来自两个或者多个表的行结合起来

    (1)left join,right join,inner join,outer join

    (2)inner join:表中至少存在一个匹配是返回行

    (3)left join:即使右表无匹配，也返回左表中所有行

    (4)right join:即使左表无匹配，也返回右表中所有行

    (5)full outer join:只要左表或者右表其中一个存在匹配，就返回行

    select column_name(s) from table1 inner join table2 on table1.column_name=table2.column_name;

  8. union:合并两个或者多个select语句的结果

    (1)select column_name(s) from table1 union select column(s) from table2;

    (2)默认选取不重复，若要允许重复，则使用 union all

  9. select into:从一个表复制数据插入到另一个表中

    (1)select * into newtable [in externaldb] from tabl1;

    (2)select column_name(s) into newtabel [in externaldb] from table1;

  10. insert into select:从一个表复制数据插入到另一个表中

    (1)insert into table2 select * from table1;

    (2)insert into table2 (column_name(s)) select column_name(s) from table2;

  1. Aggregate函数

  • AVG() - 返回平均值:

    (1)select avg(column_name) from table_name;

  • COUNT() - 返回行数

    (1)select count(column_name) from table_name;

    (2)select count(*) from table_name;

    (3)不重复:select count(distinct column_name) from table_name;

  • FIRST() - 返回第一个记录的值:select first(column_name) from table_name;

  • LAST() - 返回最后一个记录的值:selsect last(column_name) from table_name;

  • MAX() - 返回最大值:select max(column_name) from tabel_name;

  • MIN() - 返回最小值:select min(column_name) from table_name;

  • SUM() - 返回总和:select sum(column_name) from table_name;

  1. Scalar函数
  • UCASE() - 将某个字段转换为大写:select ucase(column_name) from table_name;对于sql server:select upper(column_name) from table_name;
  • LCASE() - 将某个字段转换为小写:select lcase(column_name) from table_name;对于sql server:select lower(column_name) from table_name;
  • MID() - 从某个文本字段提取字符，MySql 中使用:select mid(column_name,start[,length]) from table_name;
参数 描述
column_name 必需。要提取字符的字段。
start 必需。规定开始位置（起始值是 1）。
length 可选。要返回的字符数。如果省略，则 MID() 函数返回剩余文本。
  • SubString(字段，1，end) - 从某个文本字段提取字符
  • LEN() - 返回某个文本字段的长度:select len(column_name) from table_name;对于mysql:select length(column_name) from table_name;
  • ROUND() - 对某个数值字段进行指定小数位数的四舍五入:select round(column_name,decimals) from table_name;
参数 描述
column_name 必需。要舍入的字段。
decimals 必需。规定要返回的小数位数。
  • NOW() - 返回当前的系统日期和时间:select now() from table_name;
  • FORMAT() - 格式化某个字段的显示方式:select format(column_name,format) from table_name;

  1. group by:用于结合聚合函数，根据一个或多个列对结果集进行分组

    (1)select column_name,aggreate_function(column_name) from table_name where column_name operator value group by column_name;

  2. having:WHERE 关键字无法与聚合函数一起使用。

    HAVING 子句可以让我们筛选分组后的各组数据

    (1)select column_name,aggregate_function(column_name) from table_name where column_name operator value group by column_name having aggregate_function(column_name) operator value;
