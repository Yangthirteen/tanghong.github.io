Git操作学习

Author：tanghong | Date: 2019-07-11 | Category：

1.Git安装基本配置：
(1)配置用户和邮箱(-global参数，表示你的机器上所有的Git仓库都会使用这个配置)
$ git config –global user.name “Your Name”
$ git config –global user.email “email@example.com” P.S.也可以对不同的仓库指定不同的用户名和邮箱

2.创建本地仓库
(1)即创建一个目录，这个目录中的所以文件都被git管理而且，不管你做什么操作都会被记录，包括：增加、删除、修改文件等，都会被记录下来，以便后来跟踪与修改相关记录，甚至被还原。
(2)步骤：创建目录->进入目录->git init
(3)向版本库中增加文件，需要注意的是，在版本库中只能跟踪和管理文本文件，比txt文件、js文件、php文件、java文件等，所有的程序代码都可以的。但是像视频、图片等这些二进制文件，虽然能由git管理，但是只能记录大小，无法跟踪具体修改了什么
(4)增加文件 git status->git add->git commit->git status
git status 命令:查看文件状态
git add 命令：告诉git，我们要把什么文件提交到仓库中去，没有任何提示。那就说明我们提交完成了
git commit 命令：将文件提交到版本库中，其中的参数-m后面输入的是本次提交的版本说明，可以输入任意内容
