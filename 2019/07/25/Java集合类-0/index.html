一、Java集合类基本概念

  1. 集合类主要负责保存、盛装其他数据，因此集合类也被称为容器类。所以的集合类都位于java.util包下，后来为了处理多线程环境下的并发安全问题，java5还在java.util.concurrent包下提供了一些多线程支持的集合类。

二、Java集合类架构层次关系

Iterator

  1. 所有的集合类都实现了Iterator接口，这是一个用于遍历集合中元素的接口.

  2. 主要方法：

    (1)hasNext()是否还有下一个元素；

    (2)next()返回下一个元素；

    (3)remove()删除当前元素。

Collection

  1. Set(无序，不能重复)

(1)HashSet

(2)LinkedHashSet

(3)TreeSet

  1. List(有序，可重复)

(1)ArrayList

(2)LinkedList

(3)Vector(已过时)

Map(键值对，键唯一，值不唯一)

  1. HashMap

  2. LinkedHashMap

  3. TreeMap
